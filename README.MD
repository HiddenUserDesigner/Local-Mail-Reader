# 🤖 Asistente de IA Local para Lectura y Análisis de Correos

Este proyecto es un agente de IA que corre de forma local. Conecta la API de Gmail para extraer correos y utiliza LM Studio para analizarlos de forma privada.

## 🚀 Requisitos
* Python 3.10+
* LM Studio configurado con ia local de preferencia
* Lm Studio ejeutando como servidor al momento de hacer uso del programa


## 🛠️ Instalación y Uso

1. Instalar dependencias con `pip install -r requirements.txt`.
2. Ejecutar `python main.py`.
3. Ajustar archivos, parametros y requerimientos para cada caso de uso
